1) Стрелочные функции

array_map(function (User $user) {
    return $user->id;
}, $users)

array_map(fn(User $user) => $user->id, $users) 


2) Типизированные свойства (RFC)
class Bar
{
    public string $name;
    public ?int $amount;
    public Foo $foo;
}

3) Присваивающий оператор объединения с null (RFC)

Вместо такой длинной записи:
$data['date'] = $data['date'] ?? new DateTime();

Теперь можно будет написать так:
$data['date'] ??= new DateTime();


4) Оператор распаковки в массивах (RFC)

Теперь можно использовать оператор распаковки в массивах:

$arrayA = [1, 2, 3];
$arrayB = [4, 5];
$result = [0, ...$arrayA, ...$arrayB, 6 ,7];
<i>// [0, 1, 2, 3, 4, 5, 6, 7]</i>

5) Короткий открывающий тег <? устарел и будет удалён в PHP 8. Короткий тег <?= (echo) не пострадал.
